# VIVERSE Portals (Verified)

## Creator and App Management

- **Studio**: [https://studio.viverse.com/](https://studio.viverse.com/)
  - manage apps, review state, configs (auth/leaderboard)
- **Create**: [https://create.viverse.com/](https://create.viverse.com/)
  - world creation entry portal

## Identity

- **Avatar**: [https://avatar.viverse.com/](https://avatar.viverse.com/)
  - avatar and profile customization

## Discovery

- **Worlds Explore**: [https://worlds.viverse.com/](https://worlds.viverse.com/)
  - browse and launch worlds

## URL Patterns

- Live/preview world base: `https://worlds.viverse.com/<hub_sid>`
- Preview query variant: `https://worlds.viverse.com/<hub_sid>?preview`

## Usage Notes

- Use Studio for App ID lookup and review/submission steps.
- Use Worlds URL from CLI publish output for validation.
